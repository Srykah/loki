template<typename ...Args>
size_t loki::hash::combineHashSym(const Args&... args)¶

Hashes all arguments and combines them symmetrically.

In opposition to combineHash, combineHashSym doesn’t care about order between the arguments.

Return

A hash combination of the arguments where order doesn’t matter
